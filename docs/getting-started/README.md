# Getting Started

This directory is the shortest path to getting Mycelium installed and wired
into the wider ecosystem.

Start here:

- [installation.md](installation.md): install the binary and verify the right
  package is on your path
- [ecosystem-setup.md](ecosystem-setup.md): run shared ecosystem setup and
  repair Mycelium-owned guidance surfaces

Then use the main docs index at [../README.md](../README.md) for commands,
architecture, analytics, and troubleshooting.
